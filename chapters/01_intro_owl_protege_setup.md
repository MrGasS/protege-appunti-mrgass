# 1. Breve introduzione all'OWL e installazione di Protégé

### Ultimo aggiornamento del 17 Maggio 2026 alle ore 16:02

---
<p>Protégé è un applicativo per gestire ontologie in formato OWL.<br>
Ricordiamo che un’ontologia è un modello digitale che rappresenta un dominio di conoscenza tramite classi, proprietà e relazioni.</p>

<p>L'interfaccia utente di Protegé è distinta per tab, introduciamo quelle principali:
<ul>
<li>'Active Ontology' mostra l'ontologia attiva e, con essa, l'URI/IRI che la identificano in maniera univoca; 
</li>
<li>'Entities' contiene le varie sottoschede per gestire <i>Classes</i>, <i>Object</i>, <i>Data properties</i> e <i>Individuals</i>.</li>
</ul>
</p>

<p>Prima di proseguire, chiariamo i concetti fondamentali di OWL:
<ol>
<li><b>Classes</b> (classi): una <b>Persona</b> è una classe;</li>
<li><b>Properties</b> (proprietà) si dividono in 
<ul>
<li><b><i>Object Properties</i></b>: designa una relazione tra classi, per esempio, <b>hasParent</b> (haUnGenitore);</li>
<li><b><i>Data Properties</b></i>: attributi di una classe, per esempio, <b>hasAge</b> (haEtà) legato a un numero;</li>
</ul>
<li><b>Individuals</b> (individui): è un'istanza specifica di una classe, per esempio, <b>MarioRossi</b> è un individuo della classe <b>Persona</b>.</li>
</ol>
</p>
